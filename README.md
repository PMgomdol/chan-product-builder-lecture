# 로또 번호 생성기 (Lotto Number Generator)

이 프로젝트는 사용자가 버튼 클릭 한 번으로 6개의 로또 번호를 생성할 수 있는 간단한 웹 애플리케이션입니다.

## ✨ 주요 기능

*   **번호 생성**: '번호 생성' 버튼을 클릭하면 1부터 45까지의 숫자 중 6개의 고유한 번호를 무작위로 생성합니다.
*   **시각적 표현**: 생성된 번호는 각기 다른 색상의 공 모양으로 표시되어 시각적인 재미를 더합니다.
*   **자동 정렬**: 생성된 번호는 오름차순으로 자동 정렬되어 표시됩니다.
*   **모던한 디자인**: 깔끔하고 현대적인 디자인과 함께 부드러운 애니메이션 효과가 적용되었습니다.
*   **반응형 웹**: 데스크톱과 모바일 등 다양한 화면 크기에서 최적화된 화면을 제공합니다.

## 🚀 사용 방법

1.  이 IDE의 **미리보기(Preview) 창**에서 웹사이트가 자동으로 실행됩니다.
2.  화면에 보이는 '번호 생성' 버튼을 클릭합니다.
3.  새로운 로또 번호가 생성되는 것을 확인합니다.
